MODULE 02

LangGraph Fundamentals

Core building blocks of LangGraph — model stateful, graph-based workflows for LLM applications.

  • Graph: the main container for your agent workflow
  • Nodes: Python functions that perform a single action
  • Edges: connections that define the flow between nodes
  • State: a typed dict passed through all nodes
  • Conditional edges — branching logic based on output
  • Compiling and running your first graph

⚡ Key Concepts at a Glance

StateGraph The graph class you define your agent workflow in
State (TypedDict) Shared data object flowing through every node
Node A Python function — receives state, returns updated state
Edge A directed connection between two nodes
Conditional Edge Routes to different nodes based on a condition function
Checkpointer Persists state between runs — enables memory & resume
Tool Call LLM requests to run external functions (search, code…)
ReAct Pattern Reason → Act → Observe loop for agent reasoning
